# react-kenzie-hub

Rodando localmente
Clone o projeto

```shel
  git@github.com:Victor-Novakoski/react-kenzie-hub-victor-novakoski.git
```  


Entre no diretório do projeto
```shel
  cd Kenziehub
```
  ou
  
```shel
  code .
```


Instale as dependências
```shel
  yarn
```  


Inicie o servidor
```shel
  yarn dev
```  


Stack utilizada
Front-end: React, Styled Components e TypeScript
